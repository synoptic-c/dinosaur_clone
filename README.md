# dinosaur_clone

A dinosaur clone built with OpenGL and C++

## Controls

Key   Action
Space Jump
ESC   Menu
F1    Debug Console

## Build Requirements

Requirement Version
CMake       3.20+
C++         20
OpenGL      4.6
GLFW        3.4
GLM         latest
Dear ImGui  latest
stb_image   latest

## Dependencies

Install the required libraries before building
